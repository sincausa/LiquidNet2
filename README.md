# Installation
## Windows 
```
py -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
py run.py
```

## macOS and Linux
```
python3 -m venv venv
source env/bin/activate
pip install -r requirements.txt
python3 ./run.py
```

# If you dont have venv run:
## Windows
```
py -m pip install --user virtualenv
```
## macOS and Linux
```
python3 -m pip install --user virtualenv
```
